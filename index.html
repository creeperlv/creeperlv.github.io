Scalable Relative Image

An implementation of Vector Image, the file format is based on XML and some parts of syntax are similar to XAML. Everything in SRI is relative: relative position, relative length and relative size

A .Net 5 Library is provided to perform: i) Resolve SRI file; ii) Render SRI file to bitmap image; iii) Create a Scalable Relative Image and store it to SRI file.

A .Net 5 CLI Tool is provided to perform: Resolve SRI file and render it to a bitmap image.

A .Net 5 Avalonia GUI tool is provided to perform: Edit SRI file, view the preview, render to bitmap image, view available shapes and etc.
