Get Started

First thing first

This section will help you to prepare your computer before you download, install and play What Happened to Site-13?.

Firstly, make sure your computer meets the minimal requirementï¼š

  • CPU: Intel(R) Core(TM) i7-8650U or equivalent
  • GPU: GTX 1060 Mobile or equivalent
  • Memory: 8GB
  • OS: Windows 10 or newer/ Debian 10 or newer
  • Input: Generic Keyboard and Mouse
  • DirectX: 11 or newer
  • OpenGL: Untested
  • To use external tools, you may need .Net Framework 4.5

For windows users, to run setup application, you need .Net Framework 4.5.

Installation

Note: This section is for Windows users only, Linux users just need to unzip the package to run the game.

Once you download the image file of the game (e.g.: Site-13.0.1.xx.0.iso), open it with Windows Explorer. In opend window, double-click/open setup.exe

Follow the instructions in the setup application, then, the installation is done.

Run the game

For, windows users, you just need to double-click or open Site-13.exe. For Linux users, in common situation, you just need to run Site-13, you may need manually add execute permission to this file.
