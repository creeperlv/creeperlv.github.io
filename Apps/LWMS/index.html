Overview

LWMS is short for LightWeight Managed Server.

This software is designed to implement a server that administrator can controll almost all stage of the server through pipelines and manage commands.

This server provide following pipeline process units by default:

  • DefaultStaticFileUnit - Directly send requested file.
  • DefaultStreamProcessUnit - Directly write data to the output stream.

Following pipeline units are embeded with the pipeline:

  • LogUnit - Registered before all pipeline units to log requests
  • ErrorResponseUnit - Registered after all units to send http error message and pages such as send code 404 and 404 page when all units cannot handle one request
