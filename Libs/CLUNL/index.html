Overview

CLUNL is short for Creeper Lv's Uiversal dotNet Library.

This library is designed to help myself to sovle the problem that some code will be widely used in some projects/softwares/games. This library contains:

  • Tools to operate files and folders
  • Universal Interface to operate file/stream in same ways.
  • Buffers
  • Tool to generate a random string in any length
  • Structed file operators: INI-like file, tree structure file.
  • Pipeline
